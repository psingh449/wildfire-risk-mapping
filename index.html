Abbreviations
  • FEMA: Federal Emergency Management Agency (United States).
  • NRI: National Risk Index (FEMA county-level baseline used for comparison). For wildfire: NRI ≈ EAL × SocialVulnerability × (1 − CommunityResilience).
  • AUC: Area Under the Curve (a score that summarizes a ROC curve; 0.5 = random ranking, 1.0 = perfect ranking).
  • ROC: Receiver Operating Characteristic curve (plots true-positive rate vs false-positive rate as the threshold changes).
  • MTBS: Monitoring Trends in Burn Severity (USGS/USFS historical wildfire perimeters).
Data source
  • [src:census]: decennial Census API (population / housing). https://api.census.gov/data/2020/dec/pl
  • [src:acs]: ACS API (poverty, elderly, uninsured, income, home value, internet, etc.). https://api.census.gov/data/2021/acs/acs5
  • [src:whp]: wildfire hazard product (downloaded raster; not a live API).
  • [src:hifld]: HIFLD open data (facilities/access layers). https://hifld-geoplatform.opendata.arcgis.com/
  • [src:osm]: OpenStreetMap data (roads/connectivity proxy). https://overpass-api.de/api/interpreter
  • [px:…]: “proxy path” used when a primary dataset is missing at build time (still informative, but flagged).
  • [est:county-mean]: county-mean imputation used (e.g., missing block-group median).
  • [est:tract]: tract-level value assigned down to block groups when block-group ACS is missing.
  • [missing]: both primary and proxy paths are missing → shown as —.
Data quality
  • If the building value uses an estimate (e.g., county-mean median value), the UI may show a *.
  • Some ACS tables come back as all-null at block group for a county.
  • When that happens, we use tract-level values and assign them to block groups in that tract (debug tag [est:tract]).
  • Turn on Detail to see tags like [src:acs], [est:county-mean], [missing].
Miscellaneous
  • Alpine County has a single 2020 Census block group at our chosen geography, so it appears as one polygon; within-county normalization uses a neutral value when there is no variation.
